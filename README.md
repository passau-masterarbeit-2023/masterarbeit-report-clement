# Installation

```shell
sudo dnf install "tex(dirtytalk.sty)" "tex(glossaries.sty)" "tex(hyphenat.sty)" texlive-babel-english texlive-glossaries-english "tex(comment.sty)"
```

To build the acronym, you must run the command :

```shell
make build
```
